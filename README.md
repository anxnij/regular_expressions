Приложение — учебный Android-проект на Kotlin, который показывает, как на практике работать с регулярными выражениями.
В одном экране собран многострочный текст-пример (журнал дежурного по школе), и набор кнопок, каждая из которых выполняет отдельную операцию над этим текстом: поиск слова, поиск форм слова с разными окончаниями, поиск слов с общим корнем, подсчёт длинных слов, извлечение дат и времени, разбиение текста по слову и эвристика определения, что по телефону отвечает ребёнок.

Интерфейс сделан на Views + XML, логика — в MainActivity через ActivityMainBinding. Все операции реализованы с помощью Regex (класс Regex в Kotlin) и простых текстовых функций.
